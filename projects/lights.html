About the project:

This is a personal lighting control and alarm clock system. The project had a simple start—I didn't want to get out of my cozy bed to flip the light switch before sleeping at night. To fix that, I built a motorized light switcher that I can control using an infrared remote.

Shortly after this, I realized my room was quite dark when I woke up in the morning, making it difficult to stay awake. That morning darkness inspired an alarm clock function, which lets me customize times when I want the lights to turn on through a web interface.

I made the initial remote-controlled light switcher using an ESP32 and ESP-IDF, a servo, and IR receivers/transmitters (i.e., NEC communication). The alarm clock feature entailed creating a frontend (pictured above) that controls a Firebase Realtime Database, with ESP32 using a WiFi connection and HTTP requests to read the custom alarm settings.

In the future, I hope to add an enclosure to this project or integrate a motor that can control my curtains.
